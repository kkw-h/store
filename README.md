# 社区购 Community Store Project

本项目是一个社区团购/外卖小程序系统，包含用户端、商家端小程序及PC管理后台。

## 📚 项目文档
详细的开发规划和设计文档请参考 `docs/` 目录：
- [项目开发计划](docs/项目开发计划.md)
- [产品文档](docs/产品文档.md)
- [API 规划](docs/API规划.md)
- [数据库规划](docs/db%20规划.md)

## 🏗 项目结构

- **`store-mini/`**: C端用户小程序 (Uni-app + Vue3 + TS)
- **`store-server-mini/`**: 商家端小程序 (Uni-app + Vue3 + TS)
- **`store-server-pc/`**: 商家端管理后台 (Vue3 + Vite + Element Plus)
- **`store-api/`**: 后端 API 服务 (Python FastAPI + PostgreSQL)
