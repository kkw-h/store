# 待开发接口清单

根据项目现状及常规电商系统需求，以下接口尚未实现或需进一步完善：

## 1. 公共/基础模块 (Common)

| 接口描述 | HTTP方法 | 路径 | 备注 |
| :--- | :--- | :--- | :--- |
| **文件上传** | `POST` | `/upload` | 用于商品图片、用户头像等上传 (目前仅支持外链) |
| **支付回调** | `POST` | `/payment/notify` | 接收微信支付结果通知 (核心流程，API文档已提及但未实现) |

## 2. 管理后台模块 (Admin)

| 接口描述 | HTTP方法 | 路径 | 备注 |
| :--- | :--- | :--- | :--- |
| **订单列表(管理端)** | `GET` | `/admin/order/list` | 管理员需查看所有订单 (目前仅有用户端列表) |
| **订单详情(管理端)** | `GET` | `/admin/order/detail` | 管理员查看详情 (可能包含内部备注等) |
| **数据看板** | `GET` | `/admin/dashboard` | 首页统计 (今日订单、销售额、待处理数) |
| **库存调整** | `POST` | `/admin/product/stock` | 快速调整商品库存 |

## 3. 用户端模块 (User)

| 接口描述 | HTTP方法 | 路径 | 备注 |
| :--- | :--- | :--- | :--- |
| **取消订单** | `POST` | `/order/cancel` | 用户主动取消订单 (仅限商家接单前) |
| **用户信息** | `GET` | `/user/profile` | 获取用户详细信息 (积分、余额等) |
| **更新信息** | `PUT` | `/user/profile` | 更新昵称、头像等 |
